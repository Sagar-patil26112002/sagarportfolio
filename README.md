# Sagar Portfolio

A polished personal portfolio website for Sagar, built as a responsive single-page site with modern UI and a strong professional presentation.

## Files

- `index.html` — main portfolio page
- `styles.css` — responsive styling and visual design
- `Sagar_Resume_1.5YOE_Pune.pdf` — resume linked from the site

## How to use

Open `index.html` in a browser to view the portfolio. For local development, any simple HTTP server will work.
